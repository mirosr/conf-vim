Installation
============

Sources
-------
*  `git clone http://git.gitorious.org/mirosr/conf-vim.git`
*  `git clone https://mirosr@github.com/mirosr/conf-vim.git`

Post-installation
-----------------
*  `cd ~/.vim && ./post-installation`

Update
------
*  `cd ~/.vim && ./update`

Update bundle
-------------
*  `cd ~/.vim && ./update-bundle`
